# Blinkit Sales Performance Analysis
# Project Overview
This project focuses on analyzing Blinkit’s sales performance to uncover meaningful business insights related to revenue trends, product performance, customer ratings, and visibility impact.
The objective is not just to visualize data, but to identify patterns that can support better business decisions using data analytics and visualization techniques.
# Business Objective
The primary goal of this project is to:

• Evaluate overall sales performance

• Understand how product attributes influence sales

• Identify key drivers affecting customer ratings and revenue

• Enable interactive analysis through a dynamic Power BI dashboard

This analysis answers what is happening in Blinkit’s sales data and provides direction toward why it might be happening.
# Dataset Overview
The dataset includes information related to:

  • Item categories and types
  
  • Sales values and visibility
  
  • Customer ratings
  
  • Outlet size, location, and establishment year
  
Basic preprocessing and validation were performed to ensure analytical consistency.
# Analytical Approach
• The analysis was conducted in a structured manner:

• Initial data preparation and validation

• Exploratory analysis to identify relationships and trends

• Correlation analysis to understand variable dependencies

• Interactive dashboard creation for comparative performance analysis

Each step was aligned with business objectives rather than just visual appeal.
# Tools & Technologies Used
• Microsoft Excel - Initial data review and structure validation

• Python - Exploratory analysis & correlation analysis

• Power BI - Interactive dashboard creation & KPI tracking

This combination ensures both analytical depth and business-ready presentation.
# Dashboard Preview
The Power BI dashboard provides:

• High-level KPIs for quick performance assessment

• Category-wise and outlet-wise sales comparisons

• Dynamic metric selection for flexible analysis

• Clean Blinkit-themed design for better readability
# Key Insights
• Sales performance varies significantly across item types and outlet sizes

• Item visibility shows a positive relationship with sales performance

• Customer ratings remain relatively stable across categories, indicating consistency in service quality

• Medium-sized outlets contribute a major share of total sales
# Conclusion & Recommendations
• Optimizing product visibility can enhance sales outcomes

• Focus on high-performing item categories for revenue growth

• Maintain service quality to preserve stable customer ratings

• Use outlet-level performance insights for targeted operational strategies

This analysis demonstrates how data-driven insights can support strategic decision-making in retail operations.
